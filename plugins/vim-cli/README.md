# vim-cli Plugin

vim-cli plugin -- Provides aliases and functions to drive vim from the command line

## Usage

Add to the `plugins` array.

## Provided commands (functions/aliases)

| Alias   | Description                                           |
|:-------:|:-----------------------------------------------------:|
| `viml`  | Open file `$2` vim and make line `$1` to be topmost.  |
| `vimrc` | Open `~/.vimrc`                                       |
